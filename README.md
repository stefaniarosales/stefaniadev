# StefaniaDev 🚀

Mi portafolio personal como Desarrolladora Frontend.

## 🌐 Demo

**[stefaniadev.vercel.app](https://stefaniadev.vercel.app)**

## 🛠️ Tecnologías

- **Frontend:** React, JavaScript, HTML5, CSS3
- **Estilos:** Styled Components, Tailwind CSS, Bootstrap
- **Animaciones:** Framer Motion
- **Formularios:** Formik, Yup, EmailJS
- **Gestión de estado:** Redux Toolkit
- **Despliegue:** Vercel

## 📁 Estructura

```
src/
├── components/
│   ├── about/          # Sección About
│   ├── contact/        # Formulario de contacto
│   ├── navbar/         # Navegación
│   ├── projects/       # Portafolio de proyectos
│   ├── skills/         # Habilidades técnicas y blandas
│   ├── start/          # Hero/Home
│   └── UI/             # Componentes reutilizables
├── App.jsx             # Componente principal
└── main.jsx            # Entry point
```

## 🚀 Getting Started

### Instalación

```bash
# Clonar el repositorio
git clone https://github.com/stefaniarosales/stefaniadev.git

# Entrar al directorio
cd stefaniadev

# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev
```

### Construcción para producción

```bash
npm run build
```
